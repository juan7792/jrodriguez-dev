# Professional Website

Personal website presenting my professional profile, services, and selected technical projects.

The site provides an overview of my work as a software and data engineer, including experience, technical projects, and the types of services I offer. It also serves as a central point for clients or collaborators who want to explore my background and technical capabilities.

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- next-intl (internationalization)

## Deployment

The website is deployed on **Cloudflare Pages**.
